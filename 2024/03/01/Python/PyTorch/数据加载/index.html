数据加载

1. Dataset

在 PyTorch 中，Dataset 是一个抽象类，主要用于定义如何加载和预处理数据。它是构建自定义数据集的核心模块，负责将原始数据转换成可以输入到模型中的格式。PyTorch 的 torch.utils.data.Dataset 类提供了一种结构化的方式来管理数据集。你可以通过继承 Dataset 类来创建自定义的数据集。

Dataset 类必须实现两个方法：

  1. __len__: 返回数据集中样本的数量。
  2. __getitem__: 根据索引返回数据集中的一个样本。

2. Dataloader

一旦你定义了自己的数据集，你可以使用torch.utils.data.DataLoader来实现对数据的批处理、打乱和多线程加载等操作。DataLoader接收一个Dataset实例作为输入，使数据加载变得非常高效和灵活。
