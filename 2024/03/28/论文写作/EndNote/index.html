EndNote

0. 简介

EndNote是一款文献管理软件，与SCI/SSCI等数据库兼容良好，于科研人员十分便利。

  • 检索

    • 建立个人文献图书馆
    • 从数据库检索文献并快速导入至个人文献图书馆

  • 管理

    • 管理不同来源的中英文文献
    • 创建组、去重、排序、分析、做阅读笔记等
    • 帮助查找文献全文

  • 写作

    • 撰写论文时，快速引用相关文献
    • 一键修改参考文献引用格式
    • 准备投稿前，根据引文信息推荐合适期刊

1. 导入文献

使用EndNote之前，首先需要创建个人文献库（个人图书馆），创建成功后会产出两个文件。

  • 步骤1：file→New
  • 步骤2：选择自定义存储位置

EndNote支持多种方式的文献导入，常用有以下几种：

1.1 数据库导入

  1. 在数据库中检索文献，得到检索文献列表。

  2. 勾选所需文献，在检索结果页面找到Export或Download Citations选项，中文数据库为导出。

  3. 选择Export to EndNote，或在导出的题录格式中，选择与EndNote相匹配的格式。如未显示与EndNote匹配的格式，则可选择较通用的RIS格式，并将该题录格式文件下载下来，保存至本机。

  4. 通常情况下，从外文数据库导出的题录格式为：enw格式、ris格式、ciw格式，直接双击该文件，即可直接导入EndNote。

  5. 小贴士：对于中文数据库保存下来的题录格式，通常为txt格式。则需在EndNote中选择File--Import File，选择该txt文件，并在Import Option中选择EndNote Import选项导入。

1.2 学术搜索引擎导入

  1. 在学术搜索引擎如百度学术、谷歌学术、南科学术搜索中检索文献，得到结果列表。

  2. 以百度学术为例。检索完成，点击所需文献，在文献页面找到引用，导入链接选择所用文献管理软件EndNote。

  3. 将该文献的题录信息保存至本机，为enw格式。

  4. 双击该文件，即可直接导入EndNote。

1.3 本地PDF导入

  1. 保存在本地的PDF文档可快速导入至EndNote，既可以单篇导入，也可以批量导入。

  2. 单篇导入：在EndNote中选择File-- Import File，选择需要导入的PDF文档，并在Import Options中选择过滤器为PDF，即可完成单篇PDF导入。

  3. 批量导入：在EndNote中选择File-- Import Folder，可选择连同子文件夹一起导入，或者为该文件夹新建一个分组。EndNote通常可保留至文件夹的二级目录。

  4. 小贴士：导入至EndNote的PDF文档是通过DOI号来识别的，因此，如果文档里DOI号缺失，或PDF为非矢量文档，会导致EndNote无法识别，从而在软件中显示不全。

1.4 在线检索导入

  1. EndNote有三种模式：本地模式、联机在线模式、本地和在线的混合模式。（新版20版本仅为混合模式）

  2. EndNote内嵌了许多数据库，如果需要用其联机在线检索功能的话，首先需切换至在线模式，然后在左侧方的Online Search中选择需要检索的数据库。可点击More查看更多数据库。

  3. 输入检索条件，点击Search，进行检索。

  4. 检索完成，即可直接在EndNote中查看文献题录信息。

  5. 小贴士：EndNote支持在线检索的数据库有限，并不是所有图书馆订购的数据库都可以；检索完成后，只显示满足检索条件的结果条数，无法预览文献进行导入，因此，该方法更适用于精确检索。

1.5 手动输入

  1. 手动输入适用于少量文献或特殊文献，无法直接通过以上方式导入的文献。

  2. 在EndNote工具栏中找到New Reference，创建一条新的文献。

  3. 在Reference Type选项中，可选择该文献类型，如标准、报告、专利、政府文件、多媒体等。

  4. 输入作者时，需注意一名一行，便于EndNote识别。且通常名在前姓在后，如姓在前名在后，则需要加逗号。如：John Smith / Smith, John。

  5. 输入关键词时，需一词一行，便于EndNote识别。

2. 管理文献

2.1 文献分组

一、普通组：在EndNote界面左侧的管理区，在My Groups中可以新建一个普通组，对文献进行划分。鼠标右键点击Create Group，输入名称，即可完成普通组的创建。

二、智能组：除了普通组以外，EndNote还支持创建智能组。通过智能组，可以对分组条件进行设置。智能组是虚拟组，其分组方式是动态的，在后续导入的文献中，EndNote会将符合该设置条件的文献自动分配至该组。

2.2 排序、添加显示字段

在EndNote主界面中，默认显示Author ,Year,Title,Journal等字段，鼠标点击某一字段，可将文献列表进行升序或降序排列。鼠标右键点击字段，则可看到更多字段，如Key Words,DOI, Research Notes,URL等，勾选后即可在EndNote主界面中显示。

2.3 利用EndNote下载全文

如果图书馆订购了该文献，则在校园网IP范围内可以通过EndNote来下载文献全文。选中需要下载的文献，鼠标右键点击Find Full Text，即可下载PDF全文。

3. 引用文献

3.1 插入文献

安装EndNote后，MS Office工具栏中会自动生成EndNote选项卡，可将EndNote与MS Word连接起来，从而快速插入参考文献。

方式一：鼠标光标放在Word中需要插入文献的位置，在EndNote中选择文献，点击Insert Citation，即可在Word中插入文献。

方式二：鼠标光标放在Word中需要插入文献的位置，在Word中EndNote选项卡里，选择Insert Citation，查找EndNote中的文献信息。查找完成，选中文献，点击Insert即可。

3.2 批量修改文献格式

  1. EndNote中内置了大量引文格式可供选择，在Office Word中选择Style，可点击Select another style看更多引文格式。选择所需引文格式，则所有文献格式会一键式更新。
  2. 如果在内置的引文格式无法满足需求，则可到EndNote官网搜索并下载所需文献格式。下载完成，双击该文件即可在EndNote中打开，点击File--Save as保存该引文格式。重新打开Word，即可通过Style--Select another style找到该文献格式并使用。

3.3 消除域代码

通过EndNote向Word插入参考文献，是通过域代码格式连接的，因此方可批量修改文献格式。如文档全部编辑完成，为避免出错，需要消除域代码，将原文档转换为纯文本文档。选择Convert Citations and Bibliography--Conver to Plain Text，点击确定，转换完成，即可进行下一步投稿工作。
