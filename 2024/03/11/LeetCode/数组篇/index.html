3.统计数组中的元素

错误的集合

重复的数字在数组中出现 2 次，丢失的数字在数组中出现 0 次，其余的每个数字在数组中出现 1 次。记录每个元素在数组中出现的次数，然后遍历从 1 到 n的每个数字，分别找到出现 2 次和出现 0 次的数字，即为重复的数字和丢失的数字。

数组的度

定义了一个ElementInfo结构体来存储每个元素的计数、第一次出现的位置和最后一次出现的位置。然后，它通过两次遍历数组来找出具有相同度的最短连续子数组的长度：第一次遍历用于填充结构体数组并找出数组的度，第二次遍历用于找到与该度相对应的最短连续子数组的长度。

找到所有数组中消失的数字

法一：通过一个额外数组来记录原数组中出现过的数字，然后找出未出现过的数字。

法二：把数组本身用作哈希表来标记哪些数字出现过。

数组中重复的数据

法一：利用数组本身来记录元素出现的次数，通过加上 numsSize 来标记元素出现次数。最后，再次遍历数组，找出那些超过两次出现的元素。

法二：使用了类似于循环交换的方法，通过多次迭代，将每个数字放到正确的位置，然后再次遍历数组找到重复的元素。这样，它能够在 O(n) 的时间复杂度下完成任务，同时只使用了常量额外空间。

缺失的第一个正数

对于一个长度为 N的数组，其中没有出现的最小正整数只能在[1,N+1] 中。我们对数组进行遍历，对于遍历到的数x，如果它在[1,N] 的范围内，那么就将数组中的第 x−1 个位置（注意：数组下标从 0 开始）打上「标记」。在遍历结束之后，如果所有的位置都被打上了标记，那么答案是N+1，否则答案是最小的没有打上标记的位置加 1。

4. 数组的遍历

最大连续1的个数

如果当前元素是 1，则将当前的连续 1 的个数加 1，否则，使用之前的连续 1 的个数更新最大的连续 1 的个数，并将当前的连续 1 的个数清零。

注意：遍历数组结束之后，需要再次使用当前的连续 1 的个数更新最大的连续 1 的个数，因为数组的最后一个元素可能是 1，且最长连续 1 的子数组可能出现在数组的末尾，如果遍历数组结束之后不更新最大的连续 1 的个数，则会导致结果错误。

提莫攻击

法一：通过迭代给定的时间序列，计算毒药攻击的总持续时间，同时处理攻击时间的重叠情况。每次攻击发生时判断是否处于中毒状态，从而进行不同的处理。

法二：最后一次攻击持续时间为duration，之前的每一次攻击取时间间隔和duration的最小值。

第三大的数

使用三个变量 max1, max2, 和 max3 分别表示当前数组中的最大值、第二大值和第三大值。通过遍历数组，依次更新这三个变量。在更新过程中，跳过重复元素，以确保不会将重复的值考虑在内。最终返回第三大值或者最大值（如果数组中不存在第三大值）。

三个数的最大乘积

法一：线性扫描。如果数组中全是非负数，则排序后最大的三个数相乘即为最大乘积；如果全是非正数，则最大的三个数相乘同样也为最大乘积。如果数组中有正数有负数，则最大乘积既可能是三个最大正数的乘积，也可能是两个最小负数（即绝对值最大）与最大正数的乘积。

法二：排序（小→大）。这行代码使用了 C 语言标准库中的 qsort 函数来对数组 nums 进行排序。qsort 函数的原型如下：

参数说明： - base：指向待排序数组的指针，即 nums。 - nmemb：数组中元素的个数，即 numsSize。 - size：数组中每个元素的大小，这里是 sizeof(int)，因为数组元素是整数。 - compar：是一个函数指针，指向一个用于比较两个元素的函数。这里传入的是 cmp 函数，即自定义的比较函数。

cmp 函数是用来告诉 qsort 函数如何比较数组中的元素。它返回一个整数值，当返回值小于 0 时表示第一个参数应该排在第二个参数之前，当返回值等于 0 时表示两个参数相等，当返回值大于 0 时表示第一个参数应该排在第二个参数之后。

杨辉三角

状态转移方程为：dp[i][j] = dp[i - 1][j - 1] + dp[i - 1][j]。注意：需要单独处理每行第一个和最后一个元素。returnColumnSizes 是一个指向指针的指针，可以动态分配内存来存储每一行的列数。这样做可以灵活地适应不同行数的帕斯卡三角形，并且避免了固定大小的数组。

补充：指针

杨辉三角II

法一：同上题，仅返回最后一行数据。

法二：杨辉三角是一种数学结构，由数字组成的等腰三角形，满足以下性质：

  1. 每一行的两端都是1。
  2. 每个数字等于它上方两个数字之和。

根据这些性质，可以使用组合数的性质来表示杨辉三角的每一个数字。具体地，第 n 行的第 k 个数字（索引从0开始）可以表示为组合数 C(n, k)，即从 n 个不同元素中取出 k 个元素的组合数。

组合数的计算公式为： \[ C(n, k) = \frac{n!}{k!(n-k)!} \] 其中，n! 表示 n 的阶乘，即 n 的所有正整数的乘积。而k!(n-k)!是为了去除重复计算。

利用这个公式，可以生成杨辉三角的每一行。不过在使用这个公式时，可以发现每一项的计算都可以通过前一项来进行递推，这样可以减少计算量。假设要生成第 n 行的数字，可以利用如下递推关系： \[ C(n, k) = C(n, k-1) \times \frac{n-k+1}{k} \]

首先进行转置操作，然后对每一行进行水平翻转，实现了原地旋转90度的功能。

矩阵置零

法一：

  1. 遍历整个矩阵，如果某个元素为0，则在两个标记数组中将该元素所在的行和列的对应位置置为0，以标记需要置零的行和列。
  2. 再次遍历矩阵，根据标记数组的信息，将对应行和列上的元素置为0，完成置零操作。

法二：用矩阵的第一行和第一列代替方法一中的两个标记数组，以达到 O(1) 的额外空间。但这样会导致原数组的第一行和第一列被修改，无法记录它们是否原本包含 0。因此我们需要额外使用两个标记变量分别记录第一行和第一列是否原本包含 0。

生命游戏

通过引入两个额外的状态(-1 和 2)来标记那些将要改变状态的细胞，从而在所有细胞状态更新前避免直接改变任何细胞的状态。在第一遍遍历时，它计算每个细胞周围活细胞的数量，并根据游戏规则决定细胞的命运。然后，在第二次遍历中，它将所有标记为-1的细胞变为死亡状态（0），将所有标记为2的细胞变为活状态（1），从而完成状态的更新。这种方法确保了在更新任何细胞状态之前，所有细胞的命运都已经决定，从而遵循了游戏的规则。
