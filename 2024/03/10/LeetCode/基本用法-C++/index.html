基本用法-C++

1. 数组

C++ 中的 vector 是一种序列容器，可以存储任意类型的对象。这意味着它可以存储整数、浮点数、字符串或任何其他类型的对象。vector 是一个模板类，属于标准模板库（STL）的一部分。

关键特性：

  1. 动态数组：vector 像数组一样工作，但其大小可以动态变化。你可以在运行时添加或删除元素，而不必在编译时指定其大小。

  2. 连续存储：vector 在内存中连续存储其元素，这意味着你可以使用指针或数组索引来访问元素。

  3. 随机访问：vector 支持随机访问，这意味着你可以直接通过索引访问任何元素，时间复杂度为 O(1)。

  4. 自动管理内存：vector 自动管理其元素的内存分配和释放。当 vector 的容量不足以容纳更多元素时，它会自动扩展其容量。

  5. 灵活的大小调整：你可以使用 resize() 方法来改变 vector 的大小，增加或减少元素数量，并相应地分配或释放内存。

常用函数：

  • push_back(element): 向 vector 的末尾添加一个元素。
  • pop_back(): 移除 vector 末尾的元素。
  • size(): 返回 vector 中元素的数量。
  • empty(): 检查 vector 是否为空，即不包含任何元素。
  • clear(): 移除 vector 中的所有元素，使其大小为 0。
  • at(index): 访问指定位置的元素，与使用下标操作符 [] 类似，但带有边界检查。
  • front(): 访问 vector 中的第一个元素。
  • back(): 访问 vector 中的最后一个元素。
  • insert(position, element): 在 vector 的指定位置之前插入一个元素。
  • erase(position): 移除 vector 中指定位置的元素。
  • resize(new_size): 改变 vector 的大小，如果新大小大于当前大小，会添加默认元素。
  • reserve(capacity): 分配至少能存储指定数量元素的内存空间，这可以提高 vector 添加元素的效率。
  • reverse()：反转容器中的元素。

3. 栈

在 C++ 中，stack 是一个标准模板库（STL）提供的容器，它提供了一种先进后出（Last In First Out, LIFO）的数据结构。

  • empty() ：如果栈为空返回true，否则返回false
  • size() ：返回栈中元素的个数
  • pop()：弹栈
  • push()：压栈
  • peek()/top()：预览栈顶元素:

注意：peek() 在 C++ 的 stack 中是没有定义的方法，但是 top() 方法可以实现类似的功能，返回 stack 中的顶部元素的值，而不移除它。

4. 队列

在 C++ 中，queue 是一个标准模板库（STL）提供的容器，它提供了一种先进先出（First In First Out, FIFO）的数据结构。

  • empty() ：如果队列为空返回true，否则返回false
  • size() ：返回队列中元素的个数
  • pop() ：删除队列首元素
  • push()：在队尾压入新元素
  • front() ：预览队列首元素
  • back()：预览队列尾元素

priority_queue 是 C++ 标准库中提供的一个优先队列实现。它基于堆数据结构，并提供了一组方法来处理具有优先级的元素。

  • empty()
  • size()
  • top()：获取堆顶元素
  • pop()：删除堆顶元素
  • push()：压入新元素
  • emplace()： 原地构造一个元素并插入队列

5. 字符串

5.1 基本操作

C++中的string是标准库中的一个类，用于表示和操作字符串。它提供了一系列的成员函数来进行字符串的处理，使得字符串操作更加方便和高效。

  • empty()：测试字符串是否为空
  • length() /size()： 返回字符串的长度
  • at(pos)：返回对字符串中位置 pos 的字符的引用
  • front()：返回对字符串第一个字符的引用
  • back()：返回对字符串最后一个字符的引用
  • push_back(c)：将字符 c 附加到字符串的末尾，将其长度增加一
  • pop_back()：擦除字符串的最后一个字符，从而有效地将其长度减少一个
  • insert(pos,str)：在 pos 指示的字符之前将其他字符插入到字符串中
  • erase(pos,len)：删除字符串的一部分，从而减少其长度
  • find(str,pos = 0)：在字符串中搜索由其参数指定的序列的第一次出现。当 find() 函数未找到指定子字符串或字符时，它将返回 string::npos，表示搜索失败。
  • `substr(pos=0,len)：返回一个新构造的字符串对象，其值初始化为该对象的子字符串的副本
  • isalnum()：检查所传的字符是否是字母和数字
  • isalpha()：判断一个字符是否是字母

5.2 格式化

  • sscanf(const char* str, const char* format, ...)：从一个字符串中按照指定的格式解析数据并将其存储到变量中
  • stoi(str)：将字符串转换为整数
