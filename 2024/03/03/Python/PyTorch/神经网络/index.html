3. 池化层

torch.nn.MaxPool2d 是 PyTorch 中用于执行二维最大池化操作的模块。最大池化是一种常用的降采样技术，用于减小图像或特征图的空间尺寸，同时保留最重要的特征。

这个模块会将输入的二维数据（通常是特征图）按照给定的窗口大小，在每个窗口上取最大值作为输出。在深度学习中，通常用于在卷积神经网络（CNN）的卷积层后进行特征图的降维和特征提取。

使用 MaxPool2d 可以有效地减小特征图的维度，并且在一定程度上提取出图像的重要特征，从而帮助神经网络更好地学习和泛化。以下是一个简单的示例，展示了如何在一个卷积神经网络中使用 MaxPool2d：

4. 非线性激活

4.1 ReLU

torch.nn.ReLU 是 PyTorch 中的激活函数模块，用于实现修正线性单元（Rectified Linear Unit，ReLU）激活函数。ReLU 是深度学习中最常用的激活函数之一，其数学表达式为： \[ \text{ReLU}(x) = \max(0, x) \] 这意味着对于输入 ( x )，如果 ( x ) 大于等于零，则激活函数输出与输入相同，如果 ( x ) 小于零，则输出为零。

使用 ReLU 激活函数的主要原因是它能够解决梯度消失问题，并且计算速度较快。此外，ReLU 的输出范围是 \([0, +\infty)\)，没有上限，这有助于网络的非线性建模能力。

值得注意的是，对于输入张量中的每个元素，如果元素的值小于零，经过 ReLU 激活后会变成零；如果元素的值大于等于零，激活函数会保持不变。

4.2 Sigmoid

torch.nn.Sigmoid 是 PyTorch 中的激活函数模块，用于实现 Sigmoid 激活函数。Sigmoid 函数是一种常用的非线性激活函数，其数学表达式为： \[ \text{Sigmoid}(x) = \frac{1}{1 + e^{-x}} \] Sigmoid 函数的输出范围是 \((0, 1)\)，因此可以将输入的任意实数映射到一个范围在 0 到 1 之间的数值，常用于二分类问题中的输出层，表示样本属于某一类别的概率。

5. 线性层

torch.nn.Linear 是 PyTorch 中用于创建线性（全连接）层的模块，也称为仿射变换。线性层将输入张量与权重矩阵相乘，然后加上偏置向量，得到输出张量。

线性层的数学表达式为： \[ \text{output} = \text{input} \times \text{weight}^T + \text{bias} \] 其中： - \(\text{input}\)是输入张量，形状为\((N, \text{in\_features})\)，其中\(N\)是批量大小，\(\text{in\_features}\)是输入特征的数量。 - \(\text{weight}\)是权重矩阵，形状为(out_features, in_features)，其中\(\text{out\_features}\)是输出特征的数量。 - \(\text{bias}\)是偏置向量，形状为\((\text{out\_features})\)。

在 PyTorch 中，torch.nn.Linear 的构造函数包含两个参数：in_features 和 out_features，分别用于指定输入特征的数量和输出特征的数量。

6. Sequential

torch.nn.Sequential 是 PyTorch 中的一个容器模块，用于构建神经网络的序列模型。它可以按顺序组合多个神经网络层（如线性层、激活函数、池化层等），构建一个完整的神经网络模型。
