Transforms

PyTorch 的 transforms 是 torchvision 模块中的一部分，主要用于对图像数据进行预处理和增强，尤其在图像分类和目标检测等任务中非常常用。transforms 提供了丰富的图像变换功能，比如缩放、裁剪、旋转、归一化等。

2. 常见操作

2.1 transforms.ToTensor()

将图像从 PIL 格式或 numpy 数组转换为 PyTorch 张量，并将像素值从 [0, 255] 归一化为 [0, 1]。

2.2 transforms.Resize()

调整图像大小，接受单个整数（最小边的大小）或 tuple（宽度，高度）。

2.3. transforms.CenterCrop()

从图像中心裁剪指定大小的区域。

2.4. transforms.RandomHorizontalFlip()

以一定概率（默认 0.5）水平翻转图像，用于数据增强。

2.5. transforms.RandomRotation()

随机旋转图像一定的角度范围，用于数据增强。

2.6. transforms.Normalize()

对图像进行标准化操作，一般用于将像素值转换为零均值单位方差。它接受均值和标准差列表。如果使用了 transforms.ToTensor()，要特别注意输入的均值和标准差应匹配 [C, H, W] 格式。

# 3. 组合 transforms transforms.Compose() 用于将多个变换操作串联起来。例如，先调整大小、再转为张量、最后标准化图像。
