2. 卷积层（Convolution Layers）

torch.nn.Conv2d 是 PyTorch 中用于执行二维卷积操作的模块，常用于处理图像数据。它是卷积神经网络（CNN）中的一个关键组成部分。

2.1 主要参数

  1. in_channels: 输入数据的通道数，例如，RGB 图像有三个通道。
  2. out_channels: 卷积层输出的通道数，即卷积核的数量。
  3. kernel_size: 卷积核的大小，可以是单个整数或元组，例如 (3, 3) 表示 3x3 的卷积核。
  4. stride: 卷积的步幅，可以是单个整数或元组，决定卷积核在输入数据上移动的步长。
  5. padding: 输入数据的边缘填充，确保输出大小符合要求。可以是单个整数或元组。
  6. dilation: 卷积核元素之间的间距，用于扩张卷积。
  7. groups: 控制输入和输出通道的分组，有助于实现深度可分离卷积。
  8. bias: 是否添加偏置项，默认为 True。

3. 池化层（Pooling layers）

torch.nn.MaxPool2d 是 PyTorch 中用于执行二维最大池化操作的模块，常用于图像处理和卷积神经网络（CNN）中。最大池化有助于减少特征图的尺寸，从而降低计算量和防止过拟合，同时保留重要的特征。

3.1 主要参数

  1. kernel_size: 池化窗口的大小，可以是单个整数或元组，例如 (2, 2) 表示 2x2 的窗口。
  2. stride: 池化操作的步幅，决定窗口在特征图上移动的步长。默认为 kernel_size 的值。
  3. padding: 输入特征图的边缘填充，通常为 0。
  4. dilation: 池化窗口元素之间的间距，默认为 1。

4. 非线性激活（Non-linear Activations）

4.1 ReLU

torch.nn.ReLU 是 PyTorch 中用于实现 ReLU（Rectified Linear Unit）激活函数的模块。ReLU 是一种常用的非线性激活函数，主要用于神经网络中，引入非线性特性，使得模型能够更好地学习复杂的函数。数学表达式为： \[ \text{ReLU}(x) = \max(0, x) \] 这意味着对于输入 ( x )，如果 ( x ) 大于等于零，则激活函数输出与输入相同，如果 ( x ) 小于零，则输出为零。使用 ReLU 激活函数的主要原因是它能够解决梯度消失问题，并且计算速度较快。此外，ReLU 的输出范围是 \([0, +\infty)\)，没有上限，这有助于网络的非线性建模能力。下面是一个简单的示例展示了如何在 PyTorch 中使用 torch.nn.ReLU：

4.2 Sigmoid

torch.nn.Sigmoid 是 PyTorch 中实现 Sigmoid 激活函数的模块。数学表达式为： \[ \text{Sigmoid}(x) = \frac{1}{1 + e^{-x}} \] Sigmoid 函数的输出范围是 \((0, 1)\)，因此可以将输入的任意实数映射到一个范围在 0 到 1 之间的数值，常用于二分类问题中的输出层，表示样本属于某一类别的概率。以下是一个简单的示例展示了如何在 PyTorch 中使用 torch.nn.Sigmoid：

5. 线性层（Linear Layers）

torch.nn.Linear 是 PyTorch 中用于实现线性变换（全连接层）的模块。它将输入的特征通过一个权重矩阵进行线性变换，并可选择性地添加偏置。

线性层的数学表达式为： \[ \text{output} = \text{input} \times \text{weight}^T + \text{bias} \] 其中： - \(\text{input}\)是输入张量，形状为\((N, \text{in\_features})\)，其中\(N\)是批量大小，\(\text{in\_features}\)是输入特征的数量。 - \(\text{weight}\)是权重矩阵，形状为$ ( , )\(，其中\)$是输出特征的数量。 - \(\text{bias}\)是偏置向量，形状为\((\text{out\_features})\)。

在 PyTorch 中，torch.nn.Linear 的构造函数包含两个参数：in_features 和 out_features，分别用于指定输入特征的数量和输出特征的数量。以下是一个简单的示例，演示了如何在 PyTorch 中使用 torch.nn.Linear：
