GitHub+PicGo搭建免费图床

1. 操作步骤

1.1 Github

  • 新建仓库（注：必须为Public，勾选Add a README file）

  • 生成token：点击Github右上角的用户头→Settings→Developer settiings→Personal access tokens→Tokens(classic)→Generate new token，在Note一栏输入创建的仓库名称，并勾选repo，最后在最下方点击Generate token按钮，就生成了对应的Token。注：新生成的 Token只会显示一次，请妥善保存，如有遗失，重新生成即可。

1.2 PicGo

  • 下载PicGo所需版本

  • 安装（略）

  • 配置

    • 为显示简捷，仅勾选GitHub

    • 特别注意：设定仓库名为账户+仓库名；自定义域名需要自己有域名才填写

    • 点击设为默认图床，点击确定

1.3 Typora

文件→偏好设置→图像

2. 使用方法

Typora：格式→图像→上传所有本地图片
