1 序列

在 Python 中，序列是指包含一系列元素的数据结构。Python内置的五个常用序列结构：列表、元组、集合、字典、字符串。

1.1 索引

在 Python 中，序列（比如列表、元组、字符串等）中的元素可以通过索引访问。索引用于定位序列中特定位置的元素，索引值从 0 开始。

1.1.1 正向索引

正向索引从序列的开头（左侧）开始，使用整数值从 0 开始递增来访问元素。

1.1.2 反向索引

反向索引从序列的末尾（右侧）开始，使用负整数值从 -1 开始递减来访问元素。

1.2 切片操作

切片操作允许你获取序列中的一部分。使用 [start:stop:step] 的语法来执行切片操作。

索引和切片操作是 Python 中操作序列的重要方式，能够灵活地访问序列中的特定元素或特定范围的元素。

1.3 序列相加

3.4 元组和列表的区别

在 Python 中，元组（tuple）和列表（list）是两种常用的数据结构，它们之间有一些重要的区别：

  1. 可变性：
    • 元组是不可变的（immutable）。一旦创建了元组，其内容不能被修改，即元组中的元素不能被增加、删除或修改。
    • 列表是可变的（mutable）。列表中的元素可以被修改、添加或删除。
  2. 语法表示：
    • 元组使用圆括号 () 表示，元素之间用逗号 , 分隔。例如：(1, 2, 3)
    • 列表使用方括号 [] 表示，元素之间也用逗号 , 分隔。例如：[1, 2, 3]
  3. 性能：
    • 由于元组是不可变的，因此在特定情况下，元组的性能可能会比列表稍微好一些。因为不需要考虑元组是否会被修改，所以在某些场景下，元组的操作会更快。
  4. 适用性：
    • 元组通常用于存储不可变的数据集合，例如表示一组常量值、函数返回多个值时使用元组来打包返回值等。
    • 列表用于存储可变的数据集合，可以动态地增加、删除或修改其中的元素。
  5. 方法和操作：
    • 由于列表是可变的，它具有更多的内置方法和操作，如 append()、insert()、remove() 等，用于修改列表的内容。
    • 元组作为不可变对象，它具有更少的方法和操作。
