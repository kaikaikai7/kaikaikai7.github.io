1.1.3 文件编码声明注释

在Python 3中，默认编码为UTF-8。如果不想使用默认编码，就需要在文件的第一行声明文件的编码。

1.2 代码缩进

Python采用代码缩进和冒号区分代码之间的层次。行尾的冒号和下一行的缩进表示一个代码块的开始；而缩进结束，则表示一个代码块的结束。缩进可采用空格或Tab键实现。

1.3 编码规范

1.3.1 编写规则

  • 每个import语句只导入一个模块
  • 不要在行尾添加分号
  • 建议每行不超过80个字符（不超出屏幕大小）
  • 使用空行增加代码可读性
  • 避免在循环中使用+和+=操作符累加字符串

1.3.2 命名规范

  • 模块名尽量短小，并且全部使用小写字母，可以使用下划线分割多个字母
  • 包名尽量短小，并且全部使用小写字母，可以使用点号分割多个字母
  • 类名采用单词首字母大写形式（Pascal风格）
  • 模块内部的类采用下划线+Pascal风格的类名组成
  • 函数、类的属性和方法的命名规则同模块类似
  • 常量命名时全部使用大写字母，可以使用下划线
  • 使用双下划线开头的实例变量或方法是类私有的

2 Python中的变量

2.1 保留字与标识符

2.1.1 保留字

保留字是python中已经被赋予特定意义的一些单词，不可以把这些保留字作为变量、函数、类、模块和其他对象的名称来使用。

2.1.2 标识符

  • 即名字，主要用来标识变量、函数、类、模块和其他对象的名称
  • 命名规则：
    • 由字母、下划线和数字组成，并且第一个字符不能是数字
    • 不能使用保留字
    • 不能包含空格、@、%、$等特殊字符
    • 区分字母大小写
    • 下划线开头的标识符由特殊意义
      • 单下划线开头：不能之间访问的类属性
      • 双下划线开头：类的私有成员
      • 双下划线开头和结尾：专用标识

2.2 理解变量

在Python中，变量是用于存储数据值的标识符。它们用于引用内存中的位置，其中实际的数据存储在这些位置中。变量在程序中用于存储和操作数据，可以是数字、字符串、列表、字典等等。Python是一种动态类型语言，这意味着在给变量赋值时不需要声明数据类型，解释器会根据赋给变量的值自动推断其类型。变量名可以包含字母、数字和下划线，但必须以字母或下划线开头。

3 基本数据类型

  1. 整数（int）：用于表示整数值，例如：5, -10, 1000。
  2. 浮点数（float）：用于表示带有小数点的数字，例如：3.14, 2.71828, -0.01。
  3. 字符串（str）：用于表示文本数据，用单引号或双引号括起来，例如：'Hello', "Python", '123'。
  4. 布尔值（bool）：表示真（True）或假（False）的值，用于逻辑运算和条件判断。
  5. None类型：表示空值或缺失值，在Python中常用于表示空对象或变量。

4 数据类型转换

在Python中，你可以使用内置的函数来执行不同数据类型之间的转换。一些常用的数据类型转换函数包括：

  1. int()：将一个值转换为整数类型。

  2. float()：将一个值转换为浮点数类型。

  3. str()：将一个值转换为字符串类型。

  4. bool()：将一个值转换为布尔类型。

  5. list()：将其他数据类型转换为列表类型。

  6. tuple()：将其他数据类型转换为元组类型。

  7. dict()：将其他数据类型转换为字典类型。

这些函数可以帮助你在需要时执行数据类型之间的转换。但需要注意的是，有些转换可能会导致数据丢失或类型不兼容的情况。例如，将非数字字符串转换为整数可能会导致 ValueError。所以在进行转换时要小心确保数据的一致性。
