运算符与表达式

1 运算符

1.1 算术运算符

在Python中，有几个常见的算术运算符用于执行基本的数学运算。一些常见的算术运算符包括：

  1. 加法 +：用于相加两个数。

  2. 减法 -：用于相减两个数。

  3. 乘法 *：用于相乘两个数。

  4. 除法 /：用于相除两个数。

  5. 整数除法 //：返回除法的整数部分。

  6. 取模运算 %：返回除法的余数。

  7. 幂运算 **：用于计算一个数的幂。

这些算术运算符可以用于执行各种数学运算，你可以将它们组合使用来得到你想要的结果。

1.2 赋值运算符

赋值运算符用于将值赋给变量。它们不仅仅是把一个值赋给一个变量，还可以与其他运算符一起使用来对变量进行操作并赋值。以下是常见的赋值运算符：

  1. 等号 =：将右侧的值赋给左侧的变量。

  2. 复合赋值运算符（例如 +=, -=, *=, /=, %=, **=,//=）：将运算符右侧的值和左侧的变量进行运算，并将结果赋给左侧的变量。

  3. 多重赋值：在一行内为多个变量赋值。

  4. 链式赋值：将同一个值赋给多个变量。

这些赋值运算符能够让你更灵活地处理变量的赋值和操作，使得代码更简洁、易读。

1.3 比较（关系）运算符

比较运算符用于比较两个值，并返回一个布尔值（True或False），表示比较的结果是否成立。下面是常见的比较运算符：

  1. 等于 ==：检查两个值是否相等。

  2. 不等于 !=：检查两个值是否不相等。

  3. 大于 >：检查左侧的值是否大于右侧的值。

  4. 小于 <：检查左侧的值是否小于右侧的值。

  5. 大于等于 >=：检查左侧的值是否大于或等于右侧的值。

  6. 小于等于 <=：检查左侧的值是否小于或等于右侧的值。

这些比较运算符可用于在程序中进行条件判断，并根据条件的成立与否执行相应的代码块。

1.4 逻辑运算符

逻辑运算符用于对布尔值进行操作，可以将多个条件结合起来进行逻辑运算。Python中常见的逻辑运算符包括：

  1. 与运算 and：当且仅当所有条件都为真时，结果为真。

  2. 或运算 or：只要有一个条件为真，结果就为真。

  3. 非运算 not：用于取反一个条件的值，如果条件为真，则返回假；如果条件为假，则返回真。

这些逻辑运算符可以结合多个条件，根据这些条件的逻辑关系得出最终的布尔值结果。在条件判断、循环控制等方面，逻辑运算符非常有用。

1.5 位运算符

位运算符用于对整数在二进制形式下的位进行操作。在Python中，有几个常见的位运算符：

按位与 &

按位或 |

按位异或 ^

按位取反 ~

左移 <<

右移 >>

2 运算符优先级

在 Python 中，不同的运算符有不同的优先级。当一个表达式中有多个运算符时，Python 会根据优先级规则来确定计算顺序。以下是常见运算符的优先级，从高到低排列：

  1. 括号 ()（最高优先级）：括号中的表达式优先计算。
  2. 指数运算 **：例如，2 ** 3 表示 2 的 3 次方。
  3. 按位取反 ~
  4. 正负号 +x, -x
  5. 乘法、除法、取模、整除 *, /, %, //
  6. 加法和减法 +, -
  7. 按位左移、按位右移 <<, >>
  8. 按位与 &
  9. 按位异或 ^
  10. 按位或 |
  11. 比较运算符 ==, !=, >, <, >=, <=
  12. 逻辑运算符 not, and, or（逻辑运算符的优先级低于比较运算符）

如果在一个表达式中含有多个运算符，Python 将按照上述优先级顺序进行计算。当优先级相同时，Python 会从左向右计算表达式。为了避免混淆，建议在复杂的表达式中使用括号来明确计算顺序。
