1.1 注释规则

1.1.3 文件编码声明注释

在Python 3中，默认编码为UTF-8。如果不想使用默认编码，就需要在文件的第一行声明文件的编码。

1.2 代码缩进

Python采用代码缩进和冒号区分代码之间的层次。行尾的冒号和下一行的缩进表示一个代码块的开始；而缩进结束，则表示一个代码块的结束。缩进可采用空格或Tab键实现。

1.3 编码规范

1.3.1 编写规则

  • 每个import语句只导入一个模块；
  • 不要在行尾添加分号；
  • 建议每行不超过80个字符（不超出屏幕大小）；
  • 使用空行增加代码可读性；
  • 避免在循环中使用+和+=操作符累加字符串。

1.3.2 命名规范

  • 命名要具有描述性；
  • 必须以字母或下划线开头，且只能是下划线、字母和数字的组合；
  • 不能使用Python的关键字，也称保留字；
  • 命名是区分大小写的；
  • 模块中以下划线开头的名字是模块私有的；
  • 以双下划线开头的类成员名字是类私有的；
  • 同时以双下划线开头和结尾的名字，叫做特殊方法，也叫双下方法或魔术方法；
  • 推荐使用驼峰命名法或下划线命名法；

2. 变量

2.1 保留字与标识符

2.1.1 保留字

保留字也称关键字，不能用于命名。

2.1.2 标识符

标识符是Python中的一种符号，用于表示变量名、方法名、类名等。

2.2 理解变量

变量是存放数据值的容器。与其他编程语言不同，Python 没有声明变量的命令。首次为其赋值时，才会创建变量。

2.3 创建变量

变量不需要使用任何特定类型声明，甚至可以在设置后更改其类型。字符串变量可以使用单引号或双引号进行声明。

3. 基本数据类型

  1. 整数（int）：用于表示整数值，例如：5, -10, 1000。
  2. 浮点数（float）：用于表示带有小数点的数字，例如：3.14, 2.71828, -0.01。
  3. 字符串（str）：用于表示文本数据，用单引号或双引号括起来，例如：'Hello', "Python", '123'。
  4. 布尔值（bool）：表示真（True）或假（False）的值，用于逻辑运算和条件判断。
  5. None类型：表示空值或缺失值，在Python中常用于表示空对象或变量。

4. 数据类型转换

  1. int()：将一个值转换为整数类型。

  2. float()：将一个值转换为浮点数类型。

  3. str()：将一个值转换为字符串类型。

  4. bool()：将一个值转换为布尔类型。

  5. list()：将其他数据类型转换为列表类型。

  6. tuple()：将其他数据类型转换为元组类型。

  7. dict()：将其他数据类型转换为字典类型。
